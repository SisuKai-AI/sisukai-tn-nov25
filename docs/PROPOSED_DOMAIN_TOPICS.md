# Proposed Topics for All 90 Certification Domains

This document outlines the proposed topics for each of the 90 domains across all 18 certifications. Topics are designed to break down each domain into specific, testable knowledge areas.

**Total Estimated Topics**: ~450 topics across all domains  
**Average Topics per Domain**: 5 topics  
**Range**: 3-8 topics per domain
